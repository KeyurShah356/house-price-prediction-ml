# 🏠 House Price Prediction ML Project

This project predicts house prices using Machine Learning techniques.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 📊 Features
- Data Analysis
- Data Visualization
- Random Forest Regressor
- Feature Importance Analysis
- Price Prediction

## 🎯 Goal
To build a machine learning model capable of predicting house prices based on housing features.

## 📁 Dataset
House Prices - Advanced Regression Techniques (Kaggle)
